1️⃣ Ticket Creation Form using Bootstrap 5

Create a structured form card with the following fields:

Title – Single-line text input

Ticket Type – Dropdown select

Ticket Category – Dropdown select

Ticket Priority – Dropdown select (Low, Medium, High, Critical)

Ticket Status – Dropdown select (Open, In Progress, On Hold, Closed)

Description – Multi-line text area (minimum 4 rows)

Buttons:

Primary button: “Submit Ticket”

Secondary button: “Cancel”

Design Requirements:

Use Auto Layout (vertical spacing 16–24px)

Form width: 400–480px (centered card layout)

Clear focus states for inputs

Required field indicators (*)

Subtle border (1px neutral gray)

Consistent dropdown styling

Error and success state examples
